## Nightly Sales Reporting and Tip Calculation

This app was created as a way for HiLo Lounge in Athens, Ga to calculate and save night sale reporting

This app features 
- Day and Night sales
- Draw Counting
- Employees Tip Calculation
- PDF printout of report
